# netflix-data-analysis-powerbi-dashboard
Power BI dashboard for analyzing Netflix movies and TV shows. This project visualizes content distribution, release year trends, top countries, ratings, genres, and content types using interactive filters, KPI cards, charts, and tables for business intelligence practice.

# Netflix Movies and TV Shows Analysis Dashboard 📊🎬

## Overview

This project is a Power BI dashboard for analyzing Netflix movies and TV shows data.  
The dashboard presents insights about Netflix content, including content type, release year, country, rating, genre, and overall distribution of movies and TV shows.

The main purpose of this project is to practice data analysis, business intelligence, and dashboard development using Power BI.

---

## Project Objectives

The objectives of this project are:

- To analyze Netflix movies and TV shows data
- To compare the number of Movies and TV Shows
- To identify content trends by release year
- To explore top countries producing Netflix content
- To analyze ratings and genres
- To create an interactive Power BI dashboard for data exploration

---

## Dataset

The dataset contains information about Netflix titles.

Example features include:

- Title
- Type
- Director
- Cast
- Country
- Date added
- Release year
- Rating
- Duration
- Genre / Listed in
- Description

---

## Dashboard Features

The Power BI dashboard includes:

- KPI cards for total titles, movies, and TV shows
- Movie vs TV Show comparison
- Content trend by release year
- Top countries by number of titles
- Rating distribution
- Genre/category analysis
- Interactive filters and slicers
- Data table for detailed exploration

## Key Insights

This dashboard helps answer questions such as:

- How many titles are available in the Netflix dataset?
- What is the ratio between Movies and TV Shows?
- Which years had the highest number of released titles?
- Which countries produced the most Netflix content?
- What ratings are most common on Netflix?
- What genres appear most frequently?
